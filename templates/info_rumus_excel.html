Tutorial Menghitung Skor UTBK dengan Excel

Panduan lengkap menghitung prediksi skor UTBK menggunakan Microsoft Excel dengan rumus dan visualisasi data

Konsep Dasar Perhitungan

Skor UTBK dihitung dengan menggunakan dua rumus utama yang saling terkait:

Rumus Persentase Jawaban Benar
Persentase = (Jumlah Benar ÷ Total Soal) × 100%
Rumus Konversi ke Skor UTBK
Skor UTBK = 200 + (Persentase × 8)
Contoh Perhitungan

Jika pada subtes PPU Anda menjawab benar 15 soal dari total 20 soal:

Persentase = (15 ÷ 20) × 100% = 75%

Skor UTBK = 200 + (75 × 8) = 800

Langkah-langkah Implementasi di Excel

1

Penyiapan Struktur Tabel

Buat tabel dengan struktur kolom berikut:

Kolom Deskripsi Contoh Data
A Nama Subtes (PPU, PU, PBM, dll) PPU
B Jumlah Jawaban Benar 15
C Total Soal 20
D Persentase Benar 75%
E Skor UTBK 800
2

Input Data Awal

Masukkan data untuk setiap subtes UTBK:

Contoh Data Input
Subtes Benar Total
PPU 15 20
PU 22 30
PBM 16 20
PK 14 20
3

Rumus Persentase Benar

Di kolom D (Persentase Benar), gunakan rumus:

=B2/C2*100

Lakukan drag fill ke bawah untuk menerapkan ke semua subtes.

Catatan Penting

Pastikan format sel di kolom Persentase menggunakan format "Percentage" atau "Number" dengan 2 desimal.

4

Rumus Skor UTBK

Di kolom E (Skor UTBK), gunakan rumus:

=200+(D2*8)

Lakukan drag fill ke bawah untuk semua subtes.

5

Perhitungan Total dan Rata-rata

Di baris terakhir tabel, hitung total dan rata-rata:

TOTAL =SUM(B2:B5) =SUM(C2:C5) =AVERAGE(D2:D5) =AVERAGE(E2:E5)
Visualisasi Data

Gunakan fitur chart Excel untuk membuat visualisasi perbandingan skor antar subtes.

Tips Lanjutan Excel

1

Conditional Formatting

Gunakan conditional formatting untuk menyoroti skor tertinggi dan terendah:

=E2=MAX($E$2:$E$5) // Untuk nilai tertinggi
=E2=MIN($E$2:$E$5) // Untuk nilai terendah
2

Data Validation

Tambahkan validasi data untuk memastikan input benar:

Data → Data Validation → Whole Number → Between 0 and [Total Soal]
3

Proteksi Worksheet

Lindungi worksheet untuk mencegah perubahan rumus:

Review → Protect Sheet → Password
